## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e GitHub
- Figma

## 💻 Projeto

O DevLinks é uma página de links personalizada inspirada em um linktree, contendo:

- alternância entre modo claro e escuro
- links para redes sociais
- responsividade para mobile
- troca dinâmica de avatar com JavaScript
